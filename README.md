## 实验一操作说明

- 工作路径：`/root/py_experiment`
- 虚拟环境：`python3 -m venv /root/py_experiment/.venv`
- 安装 OpenEXR：`/root/py_experiment/.venv/bin/pip install --index-url https://pypi.org/simple openexr`

### 运行示例代码

```bash
/root/py_experiment/.venv/bin/python /root/py_experiment/experiment1.py
```

脚本输出包含：
- 鸡兔同笼求解示例
- 选择排序（升序）示例
- 冒泡排序（降序）示例
